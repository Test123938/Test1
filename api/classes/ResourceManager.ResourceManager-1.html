Class ResourceManager

The resource manager for the game engine. The resource manager interfaces with the loadable assets of a game such as images, data files, and sounds, which are all found in the dist folder. This class controls loading and updates the @reference[Scene] with the loading progress, so that the scene does not start before all necessary assets are loaded.
